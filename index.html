Rules

  1. The game starts when the user presses a key.
  2. Each level, a new color is added to the sequence and displayed to the user.
  3. The user must repeat the sequence by clicking the buttons in the correct order.
  4. If the user clicks the correct buttons in the right order, they advance to the next level.
  5. If the user clicks the wrong button, the game is over.
